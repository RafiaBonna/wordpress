Automatically create customer accounts during WooCommerce guest checkout. Boost sales while building your customer database.

== Description ==

**Smart Solution to Solve WooCommerce Guest Checkout Problems**

Stop choosing between conversion rates and customer data. Guest Checkout Account Creator automatically creates customer accounts during guest checkout, giving you the best of both worlds.

**The Problem Every Store Owner Faces:**

* Enable guest checkout → Higher sales but customers can't track orders or download products
* Require account creation → Lower conversion rates as customers abandon carts
* Manual account creation → Time-consuming and inefficient customer support

**The Solution:**

Guest Checkout Account Creator bridges this gap perfectly. Customers enjoy quick guest checkout while accounts are automatically created behind the scenes. Everyone wins!

**Perfect For:**
* Physical product stores
* Digital download shops
* Course and membership sites
* Service-based businesses
* Any WooCommerce store wanting to improve conversions while building customer relationships

== Key Features ==

* **Automatic Account Creation** - Seamlessly creates accounts during guest checkout without interrupting the process
* **Secure Password Generation** - Creates strong passwords and emails login credentials to customers
* **Instant Order Access** - Customers can immediately track orders and access downloads through their new account
* **Zero Configuration** - Works out of the box with your existing WooCommerce setup
* **Email Integration** - Sends welcome emails with login details after purchase
* **Digital Product Support** - Customers get instant access to downloads and future updates
* **Order History Access** - Full purchase history available in customer accounts
* **Support Reduction** - Dramatically reduces "where's my order" support requests

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/guest-checkout-account-creator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Settings > Guest Checkout Accounts to configure the plugin settings

== Frequently Asked Questions ==

= Does this work with all WooCommerce themes? =

Yes, this plugin is theme-independent and works with any WooCommerce compatible theme.

= Can I customize the welcome email? =

Yes, you can customize both the email subject and content from the plugin settings page. Several placeholders are available for personalization.

= Is this GDPR compliant? =

Yes, the plugin only processes data that customers have already provided during checkout and follows WooCommerce's data handling practices.
